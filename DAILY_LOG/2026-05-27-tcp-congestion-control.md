## 关键理解

| 概念 | 理解 |
|------|------|
| 流量控制 vs 拥塞控制 | 流量控制是接收方说"我满了"，拥塞控制是发送方通过丢包"猜"网络堵了 |
| 慢启动 | 指数增长（1→2→4→8），到 ssthresh 切换线性增长。叫"慢"是相对 1986 年前无拥塞控制 |
| AIMD | 加性增大（+1/RTT）+ 乘性减小（÷2），被数学证明是多流公平收敛的唯一算法 |
| Tahoe vs Reno | Tahoe 丢包砍到 1 重来；Reno 区分超时（砍到1）和 3 重复 ACK（砍一半+快速恢复） |
| RED | 路由器不等缓冲区满了才丢，提前概率随机丢，打破全局同步震荡 |
| ECN | 路由器直接在包头标记拥塞，不丢包，接收方通过 ACK 带回信号 |

## 遇到的问题
